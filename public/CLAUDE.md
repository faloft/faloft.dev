## Жёсткие границы v1

- Один скролл, не дек.
- Тёмная тема. Палитра — из плана:
  - `--bg #0F1311` (графит) / `--bg-alt #161A18`
  - `--ink #ECEDE6` / `--ink-soft #B5B8B0` / `--ink-mute #7A7E76`
  - `--accent-1 #C9A864` (золото — имя, CTA, eyebrow)
  - `--accent-2 #5FB394` (зелёный — NDA, теги, mono-числа)
- Inter (текст) + JetBrains Mono (штампы). Без других шрифтов.
- Без UI-китов, framer-motion, shadcn, tailwind-плагинов.
- Контент кейсов — рендерим из `../cases/case_*.md` через Astro Content Collections. Тексты не дублируем.

## Что НЕ делать

- Блог, FAQ, отзывы, калькулятор ROI — ни в каком виде.
- Истории-якоря (снегоход / Кумбель / медь) — это `BIO.md`, не для покупателя кейса.
- Dark/light переключатель — пользователь явно выбрал dark, не ломать.
- Не «упрощать в светлую тему по умолчанию», даже если кажется что светлая безопаснее.
- Не наращивать секции сверх пяти из плана (Hero / Что даю / Принцип AI / Кейсы / Контакты).
